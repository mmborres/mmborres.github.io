♦ Customise the game ♦

Playing is very simple - you turn over one card and then try to find a matching card. There are 5 levels to unlock. A turn that results to unmatched cards is recorded as 1 wrong move. To unlock a level, you need to complete the board. Make sure to visit the Instructions page first if you need more info on how to play the game.

PLEASE NOTE that this action takes you to the Game page and if you are not going to play right away, the default values will be used in the game. You need to visit this page to reset the values again. Unless of course you click on the browser's back and/or forward buttons. Coz it's unavoidable, right?
