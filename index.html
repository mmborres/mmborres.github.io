Memory Card Game is a browser-based game originally by GA, and then enhanced by Mabeth Borres of WDI 2019.

Two Versions of the Memory Card Game

1) Accumulating Points: Memory Game Score Top-Up ☆

This application is built on top of the base code from the pre-work module. Game rules are defined as to accumulating points and reaching maximum levels, which can be customised. Utilises HTML5, CSS and Javascript. Tested on Mac, Windows, and Android.

2) Increasing Difficulty Levels: Memory Game Level-Up ☆☆☆

Taking the base memory card game to the next level, this application uses increasing difficulty per level by adding a row of cards to the board. Unlocking a level means clearing the board of matched cards. Similarly, the number of levels can also be customised. Utilises HTML5, CSS and Javascript. Tested on Mac, Windows, and Android.
